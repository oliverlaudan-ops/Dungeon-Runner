# 🏰 Dungeon Runner

A roguelike RPG built with vanilla JavaScript and HTML5 Canvas.

## Play Now

Open `index.html` in your browser to start playing!

## Features

- **3 Character Classes**: Warrior, Mage, Rogue - each with unique skills
- **Procedural Dungeons**: Every run generates a new dungeon layout
- **10 Floors**: Increasing difficulty as you descend
- **Turn-Based Combat**: Strategic combat with enemy AI
- **Permadeath**: One life to rule them all
- **Progression**: XP, gold, and leveling system

## Controls

| Key | Action |
|-----|--------|
| W/↑ | Move Up |
| S/↓ | Move Down |
| A/← | Move Left |
| D/→ | Move Right |
| 1 | Attack |
| 2 | Defend |
| 3 | Use Skill |
| 4 | Use Item |
| 5 | Descend Stairs |
| Click | Move/Attack |
